# Laptop

**Phase:** DP-4 (Application Shell & Workspace Architecture)
**Version:** 1.0
**Status:** FOUNDATION — Permanent

---

## Viewport Range

1024px to 1279px

---

## Shell State

| Element | State |
|---------|-------|
| Header | Full variant — or compact center section |
| Sidebar | Expanded (240px) or collapsed (64px) — user preference |
| Content | Max-width constrained (reduced padding: 24px) |
| Context panel | Visible (320px), resizable. Can auto-collapse for space. |
| AI panel | Visible (320px), resizable. Auto-collapse after inactivity. |
| Footer | Visible |
| Floating panels | Limited (max 2) |

---

## Layout Rules

| Rule | Value |
|------|-------|
| Grid columns | 3 (dashboard), 2 (content), 2 (charts) |
| Typography | Full scale (slightly reduced h1 if needed) |
| Spacing | Standard padding (24px) |
| Touch targets | Minimum 44x44px |
