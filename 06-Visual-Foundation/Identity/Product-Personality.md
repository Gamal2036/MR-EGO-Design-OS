## Definition

Product Personality is how the visual identity expresses the product's character. While Brand Personality is broad, Product Personality is specific to how MR:EGO behaves as a tool in the user's hands.

---

## Personality Traits

### Attentive

The product notices. It observes user behavior, learns patterns, and adapts. Visually, this is expressed through:

- Contextual UI that appears when needed, recedes when not
- Smooth transitions that acknowledge user actions
- Stateful interfaces that remember where the user left off
- AI indicators that show the system is aware without being intrusive

### Deferential

The product never overrides. It suggests, recommends, and informs — but the user decides. Visually:

- AI outputs are visually distinct from user content (subtle badge or color shift)
- Suggestions are placed beside content, not over it
- Critical actions are visually emphasized (red for destructive)
- Non-critical AI inputs use minimal visual weight

### Organized

MR:EGO imposes structure without rigidity. Information is:

- Hierarchically structured (visual weight matches importance)
- Predictably located (same patterns across all views)
- Searchable (command palette visually prominent)
- Filterable (controls visible but not dominant)

### Proactive

The product anticipates without presuming. Visually:

- Smart defaults are pre-selected but clearly changeable
- Suggested next steps appear with low visual urgency
- Empty states guide rather than wait
- AI suggestions arrive quietly beside content, not as popups

### Honest

Transparency is visual, not just philosophical. The platform:

- Shows loading states as they are (skeleton matching final layout)
- Indicates data freshness clearly (timestamps, sync indicators)
- Labels AI-generated content explicitly
- Displays confidence levels for uncertain information

---

## Personality Spectrum by Module

| Module | Dominant Trait | Visual Expression |
|--------|---------------|-------------------|
| Dashboard | Organized | Clean grids, clear metrics, logical sections |
| Career | Proactive | Smart suggestions, progress indicators, next-step guidance |
| Documents | Deferential | Minimal chrome, focus on content, AI as assistant |
| Analytics | Attentive | Rich data views, contextual insights, trend indicators |
| Settings | Honest | Clear toggles, visible privacy controls, transparent data |
| AI Workspace | Deferential + Proactive | Suggestions beside content, thinking indicators, confidence badges |

---

## What Product Personality Is NOT

| Not This | Because |
|----------|---------|
| A salesperson | The product never pushes. It presents. |
| A nanny | The product never restricts without reason. |
| A show-off | The product never displays capability unnecessarily. |
| A mystery | The product never hides state, data, or consequences. |
| A nag | The product never reminds without value. |
