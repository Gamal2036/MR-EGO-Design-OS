# Visual Emotion

**Phase:** DP-5 (Visual Foundation)
**Version:** 1.0
**Status:** FOUNDATION — Permanent
**Design Authority:** DP-0 ([Brand-Constitution.md](../../01-Constitution/Brand-Constitution.md), [UX-Constitution.md](../../01-Constitution/UX-Constitution.md))

---

## Definition

Visual Emotion is the feeling evoked by the platform's appearance before the user reads a single word. It is the instinctive response to color, space, light, and form.

MR:EGO's visual emotion is:

**Calm competence. Quiet ambition. Warm authority.**

---

## Emotional Palette

### Calm

The dominant emotion. Users feel at ease when looking at MR:EGO.

| Visual Property | Emotional Effect |
|-----------------|-----------------|
| Generous whitespace | The platform has room to breathe, so the user can too |
| Low color saturation | No visual shouting. Colors sit at professional saturation levels |
| Soft shadows | Surfaces float gently rather than hover aggressively |
| Consistent rhythm | Predictable spacing creates a meditative reading experience |
| No flashing or pulsing | Nothing demands attention. The user chooses where to look |

### Competent

The user trusts that the platform works. This feeling comes from:

| Visual Property | Emotional Effect |
|-----------------|-----------------|
| Precise alignment | Every element in its rightful place signals engineering rigor |
| Strong hierarchy | Clear visual priority means the platform understands what matters |
| Complete states | No blank loading, no broken layouts, no awkward gaps |
| Professional typography | Well-set type signals editorial quality |
| Fast visual feedback | 50ms response time signals a system that is always ready |

### Ambitious

The platform communicates potential without aggression.

| Visual Property | Emotional Effect |
|-----------------|-----------------|
| Deep neutrals | Rich charcoal and deep slate signal substance, not timidity |
| Confident primary color | Blue is used decisively, not tentatively |
| Generous scale | Large typography where it matters signals importance |
| Forward-leaning AI glow | Subtle luminescence signals intelligence at work |
| Premium materials | Glass, depth, and light signal a tool that is ahead of its time |

## Emotional Frequencies

Emotion is not static. It shifts by context while remaining within the MR:EGO emotional range.

```
Intensity
    ^
    |   Welcome/Onboarding
    |     ● (Warm, Encouraging)
    |
    |   Dashboard / Daily Use
    |     ● (Calm, Productive)
    |
    |   Decision Support
    |     ● (Focused, Analytical)
    |
    |   AI Interaction
    |     ● (Curious, Deferential)
    |
    |   Error / Warning
    |     ● (Serious, Helpful)
    |
    |   Celebration / Success
    |     ● (Satisfied, Measured)
    |
    +-------------------------------------> Time
```

## What Users Should Feel Looking at MR:EGO

| When | Emotion |
|------|---------|
| First glance | "This looks professional and premium" |
| First interaction | "This feels responsive and polished" |
| After 5 minutes | "I can trust this platform with my career" |
| After 1 hour | "This is the most organized I have felt in weeks" |
| After 1 week | "I forget I am using software. It just works." |

---

## Emotional Anti-Patterns

| Avoid | Because |
|-------|---------|
| Urgency (red badges, pulsing) | Erodes trust, creates anxiety |
| Delight (confetti, celebrations) | Feels gimmicky in a professional tool |
| Coldness (sterile white, sharp edges) | Creates distance, reduces engagement |
| Clutter (crowded layouts) | Feels overwhelming, signals disorganization |
| Inconsistency (mixed styles) | Feels untrustworthy, signals lack of care |
| Showmanship (flashy animations) | Distracts from purpose, feels unprofessional |
