# Primary Emotion

**Phase:** DP-5 (Visual Foundation)
**Version:** 1.0
**Status:** FOUNDATION — Permanent
**Design Authority:** DP-1 ([Color-System.md](../../02-Design-Language/Color-System.md))

---

## The Primary Emotional Story

Blue is MR:EGO's primary color. Its emotional story is:

**"You can trust this. It is built with care. It will not let you down."**

Every use of primary blue in the interface reinforces this story. Every dilution of blue into decoration weakens it.

---

## Emotional Dimensions of Primary Blue

| Dimension | Description | Visual Manifestation |
|-----------|-------------|---------------------|
| Security | Blue is the color of secure systems | Used for authentication, data protection indicators |
| Clarity | Blue improves focus and concentration | Used for active states, selected items |
| Authority | Blue commands respect without aggression | Used for primary actions, key decisions |
| Intelligence | Blue is associated with analytical thinking | Used for AI branding, data visualization accents |
| Calm | Blue reduces anxiety in high-stakes environments | Used for error recovery, confirmation dialogs |

---

## When Primary Color Stimulates Positive Emotion

| Context | Emotion | Why |
|---------|---------|-----|
| User completes an action | Satisfaction | Blue confirms success |
| AI provides a suggestion | Curiosity | Blue signals intelligence at work |
| User navigates to a new section | Orientation | Blue identifies the current location |
| System processes data | Confidence | Blue indicates reliable operation |
| User hovers over interactive element | Anticipation | Blue signals something is actionable |

---

## When Primary Color Causes Negative Emotion

| Context | Negative Emotion | Why |
|---------|-----------------|-----|
| Overused (>15% of screen) | Visual fatigue | Too much blue feels overwhelming |
| Wrong shade (too bright) | Cheapness | Overly saturated blue feels unprofessional |
| No contrast with background | Confusion | Low-contrast blue is hard to distinguish |
| Used for non-interactive elements | Frustration | Users expect blue to be clickable |

---

## Primary Emotion Rules

1. **Primary blue is reserved for interactive and active elements.** Never use it for static decoration.
2. **Primary blue must maintain WCAG AA contrast** in all uses — low-contrast blue erodes trust.
3. **Primary blue on non-interactive elements is confusing** — users expect blue to be clickable.
4. **The emotional impact of blue diminishes with overuse.** Reserve it for moments that matter.
5. **Dark mode uses lighter blues** (Primary-400/500) against dark backgrounds to maintain emotional warmth.
