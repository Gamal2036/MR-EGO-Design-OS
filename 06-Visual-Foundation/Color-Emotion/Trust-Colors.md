# Trust Colors

**Phase:** DP-5 (Visual Foundation)
**Version:** 1.0
**Status:** FOUNDATION — Permanent
**Design Authority:** DP-0 ([Brand-Constitution.md](../../01-Constitution/Brand-Constitution.md))

---

## Definition

Trust Colors are the specific color combinations that build user trust. Trust is MR:EGO's most valuable asset, and color choices directly impact whether users perceive the platform as trustworthy.

---

## Trust-Building Color Combinations

### Primary Trust Pairing

| Element | Color | Emotion |
|---------|-------|---------|
| Primary button background | Primary-600 (light) / Primary-500 (dark) | Decisive action |
| Primary button text | White | Clarity, contrast |
| Primary button hover | Primary-700 (light) / Primary-600 (dark) | Commitment |

This pairing signals "this action is reliable and important."

### Content Trust Pairing

| Element | Color | Emotion |
|---------|-------|---------|
| Body text | Neutral-800 (light) / Neutral-700 (dark) | Authority |
| Background | Surface-1 (white/light gray) | Clean, prepared |
| Accent highlights | Primary-500 | Intelligence, care |

This pairing signals "this content is well-sourced and accurate."

### Data Trust Pairing

| Element | Color | Emotion |
|---------|-------|---------|
| Data value | Neutral-900 (light) / Neutral-800 (dark) | Definitive |
| Data label | Neutral-500 (light) / Neutral-400 (dark) | Supporting |
| Data background | Surface-1 | Clean foundation |

This pairing signals "this data is accurate and up to date."

---

## Trust-Eroding Color Combinations

| Combination | Why It Erodes Trust | Replacement |
|-------------|-------------------|-------------|
| Low contrast text (3:1 or below) | Hard to read = looks broken | Meet 4.5:1 minimum |
| Clashing colors (blue + green + red together) | Unprofessional, chaotic | Use one accent at a time |
| Inconsistent blues (different blues on same page) | Sloppy, uncoordinated | Use the same Primary-500 everywhere |
| Bright saturated backgrounds | Aggressive, untrustworthy | Use neutral backgrounds |
| Unlabeled semantic color | Hiding meaning = suspicious | Always pair color with label |

---

## Trust Color Rules

1. **Consistency is the strongest trust signal.** Using the same blue for the same purpose everywhere builds confidence.
2. **Contrast is a trust signal.** High-contrast text is associated with quality and care.
3. **Restraint builds trust.** Using few colors signals mastery and control.
4. **Color fidelity builds trust.** Colors that look different across screens suggest inconsistency.
5. **Predicted color behavior builds trust.** Users should know what blue means before they click.

---

## Trust Color by Context

| Context | Colors | Why They Build Trust |
|---------|--------|---------------------|
| Authentication | Primary + neutral | Security association with blue |
| Financial/career data | Deep neutral + primary | Seriousness, accuracy |
| AI recommendations | Primary + subtle glow | Intelligence, capability |
| Personal data settings | Neutral + green | Safety, control |
| Error recovery | Red + neutral + clear label | Transparency, honesty |
