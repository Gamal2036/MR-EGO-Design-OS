# Neutral Emotion

**Phase:** DP-5 (Visual Foundation)
**Version:** 1.0
**Status:** FOUNDATION — Permanent
**Design Authority:** DP-1 ([Color-System.md](../../02-Design-Language/Color-System.md))

---

## Definition

Neutral Emotion defines the emotional quality of MR:EGO's neutral color palette. Neutrals make up ~80% of every screen — they are the structural foundation of the visual language. Their emotional effect is primarily about hierarchy, readability, and calm.

---

## Neutral Color Emotional Quality

| Neutral | Emotion | Best For |
|---------|---------|----------|
| Neutral-50 | Clean slate, openness, unlimited potential | Page backgrounds, empty areas |
| Neutral-100 | Prepared surface, readiness | Card backgrounds, panels |
| Neutral-200 | Slightly raised, organized | Elevated surfaces, secondary fills |
| Neutral-300 | Gentle boundary, orderly | Borders, dividers, structural lines |
| Neutral-400 | Quiet whisper, diminished | Disabled text, subtle hints, placeholder |
| Neutral-500 | Gentle guidance, hinting | Placeholder text, secondary icons |
| Neutral-600 | Supporting voice, helpful | Secondary text, captions, metadata |
| Neutral-700 | Confident statement, clear | Body text, primary descriptions |
| Neutral-800 | Bold declaration, authoritative | Headings, section titles |
| Neutral-900 | Maximum authority, definitive | Primary headings, page titles, key data |

---

## Neutral as Emotional Foundation

Neutrals do not inject emotion the way accent colors do. Instead, they provide an emotional foundation:

| Emotional Property | How Neutrals Achieve It |
|-------------------|------------------------|
| Calm | No hue = no emotional stimulation. Neutrals allow the mind to rest. |
| Order | Consistent neutral progression (50→900) creates predictable hierarchy. |
| Clarity | High contrast between neutral levels makes structure immediately readable. |
| Professionalism | Neutrals are the default for premium tools — color is earned, not assumed. |
| Focus | Without hue competition, the user's attention goes to content, not chrome. |

---

## Neutral Light vs Dark Emotional Difference

### Light Mode Neutrals

| Quality | Description |
|---------|-------------|
| Bright | Open, airy, optimistic |
| Clean | Sterile in the best sense — pristine and prepared |
| Warm undertone | Slight warm gray (#F8FAFC), not cold blue-gray |
| Soft | Gentle transitions between levels |

### Dark Mode Neutrals

| Quality | Description |
|---------|-------------|
| Deep | Rich, substantial, solid |
| Absorbent | Colors appear to sit on the surface, not float above it |
| Cool undertone | Slight blue tint (#0F172A), creates depth |
| Comfortable | Reduced contrast prevents eye strain |

---

## Neutral Emotion Rules

1. **Neutrals should not look "dirty" or "washed out."** They should look intentional and clean.
2. **Warm neutrals (light mode) feel approachable.** Cool neutrals (dark mode) feel technical.
3. **Neutral transitions should feel logical** — the step from 100 to 200 should feel as significant as 200 to 300.
4. **Neutrals should never compete with content.** If a neutral draws attention, it is too dark or too light.
5. **The emotional goal of neutrals is invisibility** — the user should not notice the neutral palette; they should notice only the content.
