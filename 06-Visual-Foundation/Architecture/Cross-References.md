## Cross-Phase Authority

| If DP-5 and... | Conflict Resolution |
|----------------|-------------------|
| DP-0 | DP-0 always wins (constitutional authority) |
| DP-1 | DP-5 provides emotional rationale; DP-1 provides specifications |
| DP-2 | DP-5 guides application; DP-2 ensures implementation |
| DP-3 | DP-5 defines experience; DP-3 defines implementation contracts |
| DP-4 | DP-5 defines visual rules; DP-4 defines structural rules |

No conflicts exist. DP-5 is the emotional layer that sits between DP-1 (specifications) and DP-4 (structure), providing guidance for how both are applied.
