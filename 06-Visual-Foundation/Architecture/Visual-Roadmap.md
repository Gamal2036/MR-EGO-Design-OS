## Visual Foundation Maintenance

While DP-5 is permanent, minor updates may occur:

| Change Type | Approval Required | Frequency |
|-------------|------------------|-----------|
| New illustration category | Visual Design Lead | As needed |
| New icon additions | Icon consistency review | Per feature |
| Module accent color addition | Constitutional review | Per module |
| Accessibility requirement update | WCAG compliance review | Per WCAG update |
| Cross-reference update | Documentation review | As needed |

Major changes (personality, emotion, light model, depth system) require constitutional amendment.
